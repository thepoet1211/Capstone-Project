About Us

Through this capstone project we are trying to create an automatic waste segregation system which uses different sensors and technologies to do a particular task more effectively and in less time.

Why do we need an automatic waste segregating system?

An automatic waste segregation system is essential to address the growing waste management problem in many parts of the world. The increasing amount of waste generated every day has put immense pressure on landfills, leading to environmental degradation, pollution, and health hazards.

In addition, the improper disposal of waste leads to significant economic losses due to the high cost of waste management. An automatic waste segregation system is an efficient solution that can help in segregating different types of waste, such as plastics, paper, metals, and organic waste. This makes it easier to recycle and reuse these materials and reduce the amount of waste sent to landfills.

Moreover, proper waste segregation also helps to minimize pollution and environmental degradation, while ensuring compliance with regulations. An automatic waste segregation system is, therefore, necessary for sustainable waste management and environmental protection, while also being cost-effective and contributing to a healthier and safer community.

Design

Through this capstone project we are trying to create an automatic waste segregation system which uses different sensors and technologies to do the task effectively and in less time.

Why do we need an automatic waste segregating system?

The design described is a waste segregation system that uses a conveyor belt to transport waste from an input point to a sensory frame where various segregation steps take place.

The process starts with an ultrasonic sensor that detects the presence of waste on the conveyor belt. The ultrasonic sensor sends a signal to a computing unit, which initiates the movement of the conveyor belt. The waste then travels to the first segregation station, where a proximity sensor is placed. This sensor detects the presence of metallic waste on the conveyor belt. If metallic waste is detected, a pusher attached to the upper frame segregates the metallic waste and pushes it to a separate bin. If the waste is not metallic, the conveyor belt moves the waste to the next station, which is the wet segregation station.

At this station, a moisture sensor is placed on the rail of the conveyor belt. The moisture sensor moves above the waste and checks if it is wet or not. If the waste is wet, the pusher pushes it to the wet bin. If the waste is dry, the conveyor belt sends it to the dry bin.

Iterations

The following are the two models we worked on and finalised the second one

Iteration 1

The following design of dustbin was created to segregate waste from the household level. As the dustbin concept was kept simple and user friendly while the mechanism and the working was quite amazing. The dustbin takes input from user by the help of a sliding plate on which the user will place the heap and push inside the bin. The ultrasonic sensor detects the waste and gives command to a claw which aligns itself above the certain type of waste and picks it from the heap. Proceeding to the next step, sensors starts to analyze the picked waste if it’s a metal, wet or dry also the sensors sense the waste in the following order. As a result of this the microcontroller will send signal to the servo motors attached to the rotary rod which have the three bins. After obtaining the result of the waste analysis from the sensor unit the servo motor will rotate the appropriate bin and align it adjacent to the opening. Then the claw will traverse to opening and release the waste. This process will repeat itself till the bin tray gets empty.
